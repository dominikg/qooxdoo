====== HTML Editing ======

HtmlArea is a html editing widget which is part of the framework. This widget is available as [[http://demo.qooxdoo.org/1.2/apiviewer/#qx.bom.htmlarea.HtmlArea|low-level]] and [[http://demo.qooxdoo.org/1.2/apiviewer/#qx.ui.embed.HtmlArea|UI-level]] implementation. The first targets traditional webpages / single-page applications and the latter Rich Internet Applications (RIA) as preferred usecase.

Here you can find some interesting technical info.


===== Demo =====

[[http://demo.qooxdoo.org/1.2/demobrowser/#widget~HtmlArea.html|{{:documentation:1.2:htmlarea_screenshot_1_0.png?567x289|HtmlArea in qx 1.2 screenshot}}]]


===== Setup =====

One important step is necessary to get the HtmlArea up and running. 

<note>
If you setup the component without handing the **source** parameter you have to place a **blank.html** file next to your applications **index.html**.
</note>

This is necessary due the [[http://taossa.com/index.php/2007/02/08/same-origin-policy/|Same-Origin Policy]] implemented by most browsers.
